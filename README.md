# Advent of Code 2022

My solutions for Advent Of Code 2022.

## How to install Deno?
```sh
curl -fsSL https://deno.land/x/install/install.sh | sh
```

## Checking Deno is installed
```sh
deno --version
```
## How to run the code?
```ts
deno run --allow-read --allow-write --watch ./[day]/[a/b].ts
```
